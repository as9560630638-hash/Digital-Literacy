## Project Overview
 
This repository contains my Digital Literacy Portfolio submitted as part of the CSE0001 course at VIT Bhopal University. As a Student Digital Ambassador, I have completed five tasks covering digital literacy awareness, professional portfolio building, coding platforms, email etiquette, and cybercrime awareness.
